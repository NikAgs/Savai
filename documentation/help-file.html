Getting Started

Unzip the archive and find out the folder HTML. There are two templates in this folder ( Ios and Android).

  1. /css — folder with css files.
  2. /fonts — folder with icon fonts.
  3. /images — folder with image files.
  4. /js — folder with Javascript files.
  5. index.html — page demo.
  6. sendmail.php — contact form settings.

All files and folder organized nicely.

Change Color

Check <head> part of the index.html. You will find all colors link available. gold color is active. So if you want to use any other color just replace the "gold" with that color name.

Contact Form

Contact form will use php to send emails. So just open sendmail.php file. Change the current email support@mixdesigns.net to your email address and Wava email subject to your subject. Your form is ready.

(NOTE: Contact form require web server to send email. Emails can go to your spam/junk folder.)
 
 

Subscription Form

Default: Mailchimp

Current active subscribe form will save emails to mail chimp list. 

 

How to connect mail chimp list? 

Open script.js and find following script. Just replace that mailchimp form action url with your mailchimp list url

 

How to find mailchimp form action URL?

Just go to your mailchimp account. Then your list. Then Signup forms. Select  Embedded forms. You will find form action URL in this page.
